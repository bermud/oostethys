Javascript SOS Parser

IOOS DIF and SWE SOS versions are supported.

jQuery.js 1.5 is required

There are two separate Javascript modules.
  • Parses SOS GetCapabilities XML response and returns valid SOS GetObservation requests urls.
    • Source: jquery.parseSOSGetCap.js
    • Documentation.
  • Parses SOS GetObservation XML response and displays results in html table, CSV or JSON string.
    • Source: jquery.parseSOSGetObs.js
    • Documentation.
  • Examples
    • This pages demonstrates using both modules with locally cached SOS XML responses.

Note: There are a lot of SOS variations and this is not a completely general SOS parser. It has been tailored specifically to the Ocean Observing community which has been using both SWE 1.0 and NOAA IOOS DIF based XML formats. It probably does O.K. on the GetCapabilities parsing, where there is much less variablility.
